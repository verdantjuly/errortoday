# ErrorToday

Calculate and shows most occured error during last 1 day.

## Versions
- ruby 3.1.4p223
- Rails 7.1.3.2

## How to Run

- Bundle Install

```
bundle install
```

- Run Server

```
bin/rails server
```

- Register Cron Job

```
whenever --update-crontab
```
